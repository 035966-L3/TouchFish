# 社区行为守则

## 我们的承诺

我们致力于让这个社区成为一个友好、安全且公平的地方，让每位成员都能自在参与。

我们珍视并尊重每个人的独特性，无论其背景、经历或观点如何。我们鼓励真诚、友善的交流，并期待所有成员共同维护一个互相支持的环境。

## 我们鼓励的行为

我们的社区由不同背景的成员组成，我们理解彼此的沟通方式可能存在差异。因此，我们提倡以善意和尊重为出发点进行互动，并努力践行以下行为：

1.  **尊重社区**：理解并尊重本社区的初衷、活动与交流方式。
2.  **友善诚实**：以真诚和善意的态度与他人交流。
3.  **包容开放**：尊重不同的观点与经历，乐于倾听。
4.  **承担责任**：对自己的言行及贡献负责。
5.  **良性反馈**：以得体的方式提出并接受建设性意见。
6.  **弥补过失**：如果无意中造成伤害，愿意主动道歉并尽力弥补。
7.  **贡献力量**：采取其他有益于社区整体氛围与发展的行动。

## 我们不鼓励的行为

为了维护社区环境，以下行为被视为不恰当。实施、威胁实施或宣扬这些行为，均违反本守则。

*   **骚扰**：在对方明确表示不希望接触后，仍进行不必要的关注或联系。
*   **人身攻击**：对社区成员或群体进行侮辱、贬低或恶意指责。
*   **歧视与刻板印象**：基于他人的身份或固有特征进行负面评判。
*   **不当言行**：在社区场景下做出令人不适的亲密举动或言论。
*   **泄露隐私**：未经允许分享或利用他人的个人或私密信息。
*   **煽动伤害**：对任何人或群体实施、煽动或威胁施加暴力及其他伤害。
*   **损害社区**：任何其他威胁社区福祉与和谐的行为。

### 其他注意事项

*   **身份冒用**：禁止冒充他人，或以虚假身份参与社区活动。
*   **尊重原创**：分享他人成果时，请正确标明来源。
*   **谨慎宣传**：请勿以刷屏、私信轰炸等不符合社区氛围的方式进行商业推广。
*   **负责传播**：分享内容时，请以负责任的态度处理其中可能涉及的不当信息。

## 如何反馈问题

即使大家本着善意相处，有时也可能产生误解或矛盾。并非所有问题都涉及违反本守则，但及时沟通有助于解决问题。

如果你遇到或目睹了可能违反本守则的行为，请通过以下方式之一向我们反馈：
*   在项目的 Issue 中提出。
*   在社区的 QQ 群中联系管理员。
*   在洛谷等主要平台私信联系维护者。

**请注意**：如果被反馈的对象是社区维护者之一，为避免可能的干扰，建议你私聊其他维护者进行举报。

我们会认真对待每一份反馈，并在保护当事人隐私的前提下，尽快了解情况。处理过程会尽可能透明，但具体的沟通和调解通常会在私下进行，除非各方同意公开。

## 问题处理与解决

若确认存在违反本守则的行为，我们将主要本着教育、沟通和修复关系的原则进行处理，并根据情况的严重程度采取相应措施。处理方式可能包括：

1.  **提醒与沟通**：对于初次或轻微的违规，通常会进行私下沟通和提醒，解释问题所在，并期待其道歉或改正。
2.  **暂时限制**：对于重复违规或稍严重的行为，可能会采取暂时限制其在特定渠道互动等措施，让其有时间冷静和反思。
3.  **临时停权**：对于多次警告无效或较严重的违规，可能会暂时中止其社区参与权限，并设定明确的恢复条件。
4.  **永久移除**：仅在极端情况下（如严重威胁社区安全、屡教不改），才会考虑永久取消其成员资格。这是最后的手段。

我们希望所有措施都能帮助成员认识问题、弥补伤害，并最终以更好的方式回归社区。社区维护者将根据具体情况，以最有利于社区整体利益的方式行使裁量权。

## 适用范围

本守则适用于社区的所有线上空间（如 QQ 群、代码仓库、TouchFish.xin 公共服务器等）。当成员在公开场合明确代表本社区时，其行为也应符合本守则的精神。
