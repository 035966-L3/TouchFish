# TouchFish - 局域网聊天解决方案

## 演示

请观看[此仓库](https://github.com/035966-L3/TouchFish-Presentation)中的 `TouchFish-Presentation.mp4` 以查看 v4 版本的用法。

## 相比于 V3 的变化

- 合并 Client 和 Server 为一个程序
- 仅提供命令行（GUI 等发行版）
- 更换协议

## 用法

1. 启动应用
2. 选择模式，默认 Client
3. （仅 Client）连接到 ip
4. 输入用户名
5. 开始聊天

应用内命令会在登录时显示，或者键入 help 查看
